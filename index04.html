CSS 폰트 연습

웹 폰트란 무엇일까요?
웹 이용자의 컴퓨터 시스템에 폰트가 설치되어 있지 않은 경우에도
웹 서버로부터 이를 제공받아 사용할 수 있는 ‘웹 전용 글꼴’을 뜻합니다.
앞서 font-family를 이용한 글꼴 적용시 잘못 입력된 글꼴이거나
시스템에 설치되어 있지 않은 글꼴인 경우에는 요소에 적용되지 않는다는 것을 설명했는데,
웹 폰트를 이용하면 그러한 문제가 발생하는 것을 방지할 수 있습니다.

    CSS에서 배경 속성은 다음과 같습니다:
  • background-color: 요소의 배경색상을 지정합니다.
  • background-image: 요소의 배경에 사용할 이미지를 지정합니다. url() 함수를 사용하여 이미지 파일의 경로를 지정할 수 있습니다.
  • background-repeat: 배경 이미지의 반복 방식을 지정합니다. repeat, repeat-x, repeat-y, no-repeat 등의 값을 사용하여 설정할 수 있습니다.
  • background-position: 배경 이미지의 위치를 지정합니다. top, bottom, left, right, center 등의 키워드를 사용하거나 픽셀 단위로 직접 위치를 지정할 수 있습니다.
  • background-attachment: 배경 이미지가 스크롤에 따라 움직일 것인지, 고정될 것인지를 지정합니다.
  • background: 위에서 언급한 속성들을 공백으로 구분하여 한 번에 지정할 수 있습니다.
  • background-size: 배경 이미지의 크기를 지정합니다. auto, cover, contain, 픽셀 단위, 백분율 등을 사용하여 설정할 수 있습니다.
  • background-origin: 배경 이미지의 기준점을 지정합니다. border-box, padding-box, content-box 등의 값을 사용하여 설정할 수 있습니다.
  • background-clip: 배경 이미지의 영역 범위를 지정합니다. border-box, padding-box, content-box 등의 값을 사용하여 설정할 수 있습니다.
